# Hello World Project

Proyek sederhana Hello World dalam **HTML** dan **Python** berlisensi **MIT**.

## Berkas Proyek

- **`index.html`**: Halaman Hello World berbasis HTML dengan tampilan modern.
- **`hello.py`**: Script Hello World dalam bahasa Python.

## Cara Menjalankan

### HTML
Cukup buka berkas `index.html` di peramban (browser) web Anda.

### Python
```bash
python3 hello.py
```
